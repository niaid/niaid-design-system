What is a Design System?

A design system is a collection of web components and visual styles that are governed by a clear set of usage standards. These patterns and styles can be used and reused to build any number of web and mobile applications, ensuring that each digital property has a familiar look and feel. Major organizations in both the public and private sectors have leveraged design systems to ensure branding consistency, reduce development and design time, and, ultimately, serve their users more efficiently.
