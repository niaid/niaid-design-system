Icons

NDS uses the Font Awesome library for all icons (with the except of third-party applications and companies). Icons are selected using classes. The specific icon is selected with the fa- class (i.e. fa-envelope). Specify the icon weight (thickness) by adding the following class corresponding to your needs:

  • fa/far - Regular
  • fal - Light
  • fas - Solid
  • fab - Brands (i.e. Facebook, Twitter, Linkedin, etc.)

Visit the Font Awesome Library to view all supported icons.
